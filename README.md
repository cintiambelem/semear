# Projeto Semear - Plataforma de Capacitação Tecnológica

Este é o projeto desenvolvido como parte da Atividade Extensionista II do curso de Análise e Desenvolvimento de Sistemas (UNINTER). A plataforma "Semear" visa a capacitação técnica de mulheres e pessoas não-binárias, com foco inicial na comunidade do Rio de Janeiro - RJ.

## 🚀 Arquitetura do Projeto
O sistema foi desenvolvido seguindo uma arquitetura de separação de responsabilidades:
- **Front-end:** Desenvolvido em React com Tailwind CSS.
- **Back-end:** API RESTful em Node.js para servir os dados.
- **Banco de Dados:** SQLite (persistência local e leve).

## 📋 Pré-requisitos
Para rodar este projeto localmente, você precisará ter instalado:
- [Node.js](https://nodejs.org/) (versão LTS recomendada)
- Git
